# Nous_dm-mc02_FEETECH_6-axis_robotic_arm

这是一个基于 **STM32H723VGT6** 控制 **FEETECH 6轴机械臂** 的固件工程，主要用于通过串口总线控制 FEETECH STS3215 舵机，实现机械臂各关节的运动控制与调试功能。

## 项目简介

本项目主要完成以下功能：

- 通过 **USART10** 与 FEETECH 舵机总线通信
- 支持 **FEETECH STS3215** 串行舵机协议
- 通过 **UART7** 输出调试信息到串口助手/PC
- 上电后可进行舵机搜索、通信测试和状态调试
- 适用于 6 自由度机械臂的基础控制与联调

## 硬件连接

### 1. 舵机总线接线（USART10）

本项目中，机械臂舵机通过 **USART10** 与主控板通信：

- **PE3（USART10_TX）** → 舵机 RX
- **PE2（USART10_RX）** ← 舵机 TX

#### 接线说明

- 主控板的 **TX 接舵机 RX**
- 主控板的 **RX 接舵机 TX**
- 舵机总线需要与主控板 **共地（GND 相连）**
- 舵机供电请按照舵机额定电压单独供电，**不要直接用 MCU 引脚供电**

### 2. 调试串口接线（UART7）

调试信息通过 **UART7** 输出：

- **PE8（UART7_TX）** → USB 转串口模块 RX
- **PE7（UART7_RX）** ← USB 转串口模块 TX

#### 调试说明

- 串口波特率通常为 **115200**
- 用于查看启动日志、搜索舵机结果、通信调试信息等

### 3. 测试注意事项

- 舵机上电前请先确认接线正确
- 若进行 **PE3 与 PE2 回环测试**，请先断开舵机，避免总线冲突
- 如果通信异常，建议先检查：
  - TX/RX 是否接反
  - 是否共地
  - 舵机供电是否正常
  - 串口参数是否一致

## 功能说明

### 1. 舵机通信

本工程支持 FEETECH STS3215 串口舵机通信，通信速率为：

- **115,200 bps**
- **8N1**

可用于向舵机发送位置、速度、扭矩等控制命令，并读取舵机状态。

### 2. 舵机搜索

程序启动后可执行舵机扫描功能，用于检测总线上有哪些舵机在线，并打印对应 ID。

### 3. 调试输出

程序通过 UART7 输出运行日志，便于：

- 查看启动状态
- 检查通信是否成功
- 调试舵机连接
- 观察搜索结果和错误信息

### 4. 机械臂控制基础

本项目可作为 6 轴机械臂控制程序的基础，后续可扩展为：

- 关节角度控制
- 多舵机同步运动
- 机械臂动作编排
- 回零与姿态控制

## 开发环境

- **下载调试器**：ST-Link

## 使用说明

1. 确认硬件接线正确
2. 给主控板和舵机上电
3. 下载固件到开发板
4. 打开调试串口查看日志
5. 观察舵机搜索和通信输出，确认机械臂工作正常

## 常见问题

### 1. 舵机无反应
- 检查舵机是否供电
- 检查 PE3 / PE2 是否接反
- 检查舵机是否与主控板共地

### 2. 串口输出乱码
- 检查 UART7 波特率是否为 115200
- 检查串口线是否接反
- 检查 USB 转串口模块是否正常

### 3. 舵机通信失败
- 检查 USART10 配置是否正确
- 检查总线速度是否为 1Mbps
- 检查舵机型号是否为 FEETECH STS3215
